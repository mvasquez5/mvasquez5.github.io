Site Name:

Weather in my Town


Site Domain:

weatherinmytown.com


Site Purpose:

The purpose of the site is to show the weather on the current user location. It will obtain user location and show the weather on his town on the current day. The user will be able to change location and query for weather on different places. It also will show weather predictions for different places and dates.
